# POM内是POI所需的对应的maven jar

```XML
	 <dependency>
         <groupId>org.apache.poi</groupId>
         <artifactId>poi</artifactId>
          <version>3.10-FINAL</version>
     </dependency>
     <dependency>
         <groupId>org.apache.poi</groupId>
         <artifactId>poi-ooxml</artifactId>
         <version>3.10-FINAL</version>
     </dependency>
     <dependency>
         <groupId>org.apache.poi</groupId>
         <artifactId>poi-ooxml-schemas</artifactId>
         <version>3.10-FINAL</version>
     </dependency>
```
